### **ðŸ“ Example: Running Lighthouse with Specific Metrics**

```javascript
        firstContentfulPaint: 2000, // FCP < 2s
        largestContentfulPaint: 2500, // LCP < 2.5s
        cumulativeLayoutShift: 0.1, // CLS < 0.1
        interactive: 4000, // TTI < 4s
      },
    });
  });
});
```
