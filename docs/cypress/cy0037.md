## **1ï¸âƒ£ Preventing New Tabs and Opening in Same Window**

### **Example: Removing `target="_blank"`**

ğŸ”¹ `.invoke('removeAttr', 'target')` â†’ Removes `target="_blank"`, forcing the link to open in the same tab.
