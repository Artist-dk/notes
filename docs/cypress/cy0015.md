### **ğŸ”¹ Selecting and Iterating Through Lists**

Tables are structured using `<table>`, `<thead>`, `<tbody>`, `<tr>`, and `<td>` tags. Cypress can **iterate through rows, validate cell values, and extract data**.

### **ğŸ”¹ Iterating Through Table Rows**

âœ… **Best Practice**: Use `cy.wrap($row)` to interact with elements inside `.each()`.
