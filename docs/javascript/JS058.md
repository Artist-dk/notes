# ğŸ“Œ **4ï¸âƒ£ Test-Driven Development (TDD)**
**TDD Process:**
1ï¸âƒ£ Write a test first (it will fail)  
2ï¸âƒ£ Write the minimal amount of code to pass the test  
3ï¸âƒ£ Refactor the code while ensuring the test still passes  

Example:
```js
// Step 1: Write a test (Failing Test)
test("should capitalize 'hello' to 'Hello'", () => {
  expect(capitalize("hello")).toBe("Hello");
});

// Step 2: Implement function
function capitalize(str) {
  return str.charAt(0).toUpperCase() + str.slice(1);
}

// Step 3: Run test again (It should pass)
```
