# ðŸš€ **JavaScript Parameters & Arguments: Default, Rest, Spread (`...args`)**  

Understanding **parameters** and **arguments** in JavaScript is crucial for writing flexible and reusable functions. ES6 introduced powerful features like **default parameters**, **rest parameters**, and the **spread operator** to enhance function behavior.

# ðŸ“Œ **1. Parameters vs Arguments**  

| Term | Definition |
|------|------------|
| **Parameter** | A **placeholder** in a function definition that receives values when the function is called. |
| **Argument** | The **actual value** passed to a function when calling it. |

### **ðŸ”¹ Example**
```js
function greet(name) {  // "name" is a parameter
    console.log(`Hello, ${name}!`);
}

greet("Alice");  // "Alice" is an argument
```

# ðŸ“Œ **2. Default Parameters (`parameter = value`)**
In **ES6**, JavaScript introduced **default parameters**, which allow you to assign default values to function parameters.

### **ðŸ”¹ Why Use Default Parameters?**
âœ” Prevents `undefined` values if an argument is missing.  
âœ” Avoids extra checks inside the function.  
âœ” Provides default behavior.

### **ðŸ”¹ Example: Default Parameter**
```js
function greet(name = "Guest") {
    console.log(`Hello, ${name}!`);
}

greet("John");  // Output: Hello, John!
greet();        // Output: Hello, Guest!
```
ðŸ”¹ If an argument is **not provided**, JavaScript uses the **default value** `"Guest"`.

---

### **ðŸ”¹ Example: Default Parameter with Multiple Values**
```js
function calculatePrice(price, tax = 0.05) {
    return price + price * tax;
}

console.log(calculatePrice(100));  // Output: 105 (uses default tax of 5%)
console.log(calculatePrice(100, 0.1));  // Output: 110 (overrides default tax)
```

---

### **ðŸ”¹ Example: Using Expressions as Default Values**
You can use expressions or even function calls as default values.

```js
function randomNumber() {
    return Math.floor(Math.random() * 10);
}

function greet(name = "Guest", id = randomNumber()) {
    console.log(`Hello, ${name}! Your ID is ${id}.`);
}

greet(); // Output: Hello, Guest! Your ID is 7.
```

# ðŸ“Œ **3. Rest Parameters (`...args`)**
Rest parameters allow a function to accept an **infinite number of arguments** as an array.

### **ðŸ”¹ Why Use Rest Parameters?**
âœ” Handles **variable-length** arguments.  
âœ” Avoids using `arguments` object (which doesn't work in arrow functions).  
âœ” Useful for mathematical operations, data aggregation, etc.

### **ðŸ”¹ Syntax**
```js
function functionName(...restParameter) {
    // Function body
}
```

### **ðŸ”¹ Example: Sum Function with Rest Parameters**
```js
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

console.log(sum(1, 2, 3));       // Output: 6
console.log(sum(5, 10, 15, 20)); // Output: 50
```
ðŸ”¹ `...numbers` collects **all arguments** into an array: `[1, 2, 3]`.

---

### **ðŸ”¹ Example: Rest Parameter in Combination with Regular Parameters**
```js
function introduce(firstName, lastName, ...hobbies) {
    console.log(`Name: ${firstName} ${lastName}`);
    console.log(`Hobbies: ${hobbies.join(", ")}`);
}

introduce("John", "Doe", "Reading", "Gaming", "Cycling");
/*
Output:
Name: John Doe
Hobbies: Reading, Gaming, Cycling
*/
```
ðŸ”¹ `...hobbies` collects the remaining arguments into an array.

---

### **ðŸ”¹ Example: Using Rest Parameters for Flexible Functions**
```js
function multiply(multiplier, ...numbers) {
    return numbers.map(num => num * multiplier);
}

console.log(multiply(2, 1, 2, 3, 4)); // Output: [2, 4, 6, 8]
```
ðŸ”¹ `multiplier` gets the first argument, while `...numbers` collects the rest.

# ðŸ“Œ **4. Spread Operator (`...args`)**
The **spread operator** (`...`) allows us to **expand** elements from an array or object into individual elements.

### **ðŸ”¹ Why Use the Spread Operator?**
âœ” Expands an **array into separate elements**.  
âœ” Copies or merges arrays & objects.  
âœ” Passes array values as function arguments.

---

### **ðŸ”¹ Example: Spread Operator in Function Calls**
```js
function sum(a, b, c) {
    return a + b + c;
}

const numbers = [3, 5, 7];

console.log(sum(...numbers)); // Output: 15
```
ðŸ”¹ `...numbers` spreads `[3, 5, 7]` into `3, 5, 7`.

---

### **ðŸ”¹ Example: Cloning Arrays**
```js
const original = [1, 2, 3];
const copy = [...original];

console.log(copy); // Output: [1, 2, 3]
```
ðŸ”¹ `...original` creates a **new copy** instead of referencing the original array.

---

### **ðŸ”¹ Example: Merging Arrays**
```js
const arr1 = [1, 2, 3];
const arr2 = [4, 5, 6];

const merged = [...arr1, ...arr2];

console.log(merged); // Output: [1, 2, 3, 4, 5, 6]
```
ðŸ”¹ Concatenates arrays without using `concat()`.

---

### **ðŸ”¹ Example: Spread Operator in Objects**
```js
const person = { name: "Alice", age: 25 };
const details = { ...person, city: "New York" };

console.log(details);
// Output: { name: "Alice", age: 25, city: "New York" }
```
ðŸ”¹ Creates a **new object** with additional properties.

---

### **ðŸ”¹ Example: Copying an Object Without Reference**
```js
const obj1 = { a: 1, b: 2 };
const obj2 = { ...obj1 };

obj2.a = 100;

console.log(obj1.a); // Output: 1 (unchanged)
console.log(obj2.a); // Output: 100
```
ðŸ”¹ **`...obj1` creates a new object** instead of modifying the original.

---

# ðŸ“Œ **5. Key Differences Between Rest and Spread**

| Feature | Rest (`...args`) | Spread (`...`) |
|---------|-----------------|---------------|
| **Purpose** | Collects multiple values into an array | Expands an array into individual elements |
| **Use Case** | Function parameters | Function calls, array/object manipulation |
| **Position** | Last parameter in function definition | Anywhere in function calls or arrays |
| **Example** | `function sum(...nums) {}` | `sum(...[1,2,3])` |

---

# ðŸš€ **Summary**
âœ” **Default Parameters** provide **fallback values** for missing arguments.  
âœ” **Rest Parameters (`...args`)** collect multiple function arguments into an array.  
âœ” **Spread Operator (`...`)** expands arrays/objects into individual elements.  
âœ” Rest is used in **function definitions**, while spread is used in **function calls & array/object manipulation**.
