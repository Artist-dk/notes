# **ðŸ“Œ 5. `switch` Statement**  
Used when multiple conditions **depend on the same variable**.  

### **ðŸ”¹ Syntax**
```js

    case value1:
        // Runs if expression === value1

        // Runs if none of the cases match
}
```

### **ðŸ”¹ Example**
```js

        console.log("Start of the week!");

        console.log("Weekend is near!");
        break;
    default:
        console.log("It's a normal day.");
}
```
âœ” Output: `"Start of the week!"` (because `day = "Monday"`).  

**ðŸ”¹ `break;` is necessary** to prevent execution from continuing into the next case.
