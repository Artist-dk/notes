### **ðŸ”¹ Example: Print Numbers 1 to 5**

# **ðŸ“Œ 4. `do...while` Loop**

### **ðŸ”¹ Syntax**


### **ðŸ”¹ Example: Print Numbers 1 to 5**

âœ” **Output:**  

### **ðŸ”¹ Key Difference Between `while` and `do...while`**

âœ” **Output:**
